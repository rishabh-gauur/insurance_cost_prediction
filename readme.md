This project uses Machine Learning to predict annual medical insurance costs. By analyzing a dataset of health factors—like age, BMI, and smoking habits—I built a regression model that identifies how lifestyle choices impact financial charges.

The  uses a Python-based pipeline to clean the data, encode categories, and scale numerical values automatically. The trained "intelligence" is saved as a .pkl file, allowing it to make instant predictions without retraining. This end-to-end workflow demonstrates how data science can transform raw health records into a professional tool for financial forecasting and risk assessment.

Problem Statement : The insurance industry relies heavily on risk assessment. Factors like smoking status, age, and Body Mass   Index (BMI) are known to significantly impact healthcare expenses. For a consumer, understanding how a change in lifestyle might affect their long-term insurance costs is valuable.

Before the model can learn, the data must be prepared. Machine learning models cannot "read" text, so categorical data like "smoker" (yes/no) and "region" must be converted into numbers. I have used encoding techniques to transform these labels.

In this project I used DECISION TREE REGRESSION to predict the cost of insurance based on the inputs.
